# 1. Практика > теория = 90% практика / 10% теория

# 2. Каждая неделя заканчивается результатом
> Результат = код.

# 3. Каждые 4 недели
  * экзамен;
  * code review;
  * рефлексия;
  * корректировка.

# Использование нейронок
  ## Разрешается ✔:
  * объяснить;
  * найти ошибку;
  * провести code review.

  ## Запрещается ❌:
  * писать решение вместо тебя.

# 5. Если проект не закончен — следующая неделя не начинается.

# График

Будни: 
* Пн–Чт - (≈1 ч. 15 мин.)
* Пятница — отдых.
* Суббота - 3 часа
* Воскресенье - 3 часа

**Критерии перехода между семестрами**

Нельзя перейти дальше, пока:

* проект не завершен;
* код не прошел ревью;
* не можешь объяснить архитектуру;
* не отвечаешь на контрольные вопросы;
* не понимаешь, почему решение устроено именно так.
